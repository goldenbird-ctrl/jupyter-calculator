# 🧮 Simple Interactive Python Calculator

This is my very first Python project, built as part of my journey into **Data Analytics**! It is a simple, interactive calculator designed specifically to run inside a **Jupyter Notebook**.

## 📊 Why I Built This (Data Analytics Perspective)
As an aspiring Data Analyst, this project helped me practice core coding foundations:
* **Jupyter Notebook:** Getting familiar with the environment where most data analysis happens.
* **Logic & Loops:** Using `while` loops and `if-elif` statements to process user inputs.
* **Error Handling:** Implementing `try-except` blocks to prevent crashes (a crucial skill when cleaning messy data later on!).

## 🚀 How to Run
1. Open the `.ipynb` file in your Jupyter Notebook.
2. Select the cell and press `Shift + Enter` to run.
3. Enter your first number, choose an operation (`+`, `-`, `*`, `/`), and enter the second number.
4. Type `exit` at any input prompt to stop the calculator.

## 🛠️ Tools Used
* **Language:** Python 3
* **Environment:** Jupyter Notebook
